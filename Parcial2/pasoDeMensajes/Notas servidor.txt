************ Descripcion sockaddr_in y htons(8080);**********************************************************

#include <stdio.h>
#include <netinet/in.h> // <- Aquí la incluimos

int main() {
    struct sockaddr_in direccion_servidor;

    direccion_servidor.sin_family = AF_INET;       // Indica que usamos IPv4
    direccion_servidor.sin_port = htons(8080);     // Asigna el puerto 8080 (convertido para la red)
    direccion_servidor.sin_addr.s_addr = INADDR_ANY; // Escucha en cualquier IP disponible

    printf("Configuración de red lista.\n");
    return 0;
}



********************  socket() ***********************************

int mi_socket = socket(dominio, tipo, protocolo);
Dominio (domain): Especifica la familia de protocolos. Lo más común es usar AF_INET (para IPv4) o AF_INET6 (para IPv6).

Tipo (type): Especifica cómo se enviarán los datos. Usas SOCK_STREAM si quieres una conexión segura y ordenada (TCP) o SOCK_DGRAM para envíos rápidos sin confirmación (UDP).

Protocolo (protocol): Generalmente se pone en 0 para que el sistema operativo elija automáticamente el protocolo correcto según el tipo (por ejemplo, TCP para SOCK_STREAM).
